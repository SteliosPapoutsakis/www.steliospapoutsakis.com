Achievements

  • Finite State Machine Verilog Code Generation (Undergraduate Research)

    I developed a Java program which through automation translated a custom high level behavioral description of an Extended Finite State Machine (FSMD) into a Verilog RTL model. The custom grammer was parsed and interpreted using Antlr. The tool supports basic arithmetic (+,-,*,/,&) of registrar values at each state, as well as branching conditionals. The tool elegantly separates the computational unit and the design controller into separate verilog modules. Additionally, I also developed a GUI using JavaFX that allows users to easily describe their FSMD.


    Github: Tool link, Gui link
  • Sign Language Glove Translator (Senoir Capstone)

    My capstone group developed a wearable glove that would classify gestures as English letters, as described in the American Sign Language (ASL). This device used flex sensors, which changed their resistive value based on how much they are flexed, in combination with an accelerometer. We used an Arduino Nano 33 BLE which would process data from the sensors. A Machine learning Neural Network "one vs rest" classifier was developed with Tensorflow and converted into a C header file using TinyML. This header file allowed us to run the model on the glove, which would continuously run and classify gestures .
